# 优化结果的结构与读取

## 返回值说明

**qt.run(op, mode=2)** 返回的对象通常为优化结果容器（如优化结果列表、Besttester 或类似结构），与 qteasy 2.0 API 一致。

## 优化结果字段完整列表（罗列并简短解释）

| 字段/属性 | 含义 |
|-----------|------|
| 每组参数 | 各次评估的参数组合（如 short_period、long_period）。 |
| 回测指标 | 该组参数对应的夏普、回撤、收益等。 |
| 目标函数值 | 用于排序的目标值（如夏普）。 |
| 排序方式 | 通常按目标函数降序（越大越优）或升序（如回撤越小越优）。 |

其他可能字段：评估耗时、约束是否满足等，以实际返回为准。

## 如何读取最优参数

从结果对象中通过属性或方法取出**最优参数组合**（如 **best_pars**、**get_best_parameters()** 等），以当前 API 文档为准。

## 如何读取 Top-K 结果

结果中通常提供多组较优参数及对应表现（如前 opti_output_count 组）；通过结果对象的列表、DataFrame 或专用方法访问，便于分析多组候选参数。
