# 回测结果评价与分析

## 绩效指标完整列表（罗列并简短解释）

以下为回测结果中常用或 **evaluate** 可用的指标，与 qteasy 2.0 一致为准。

| 指标 | 含义 |
|------|------|
| 总收益 / total_return | 区间累计收益率。 |
| 年化收益 / annual_return | 年化后的收益率。 |
| 夏普比率 / sharpe_ratio | 风险调整后收益。 |
| 最大回撤 / max_drawdown | 区间内最大净值回撤。 |
| 胜率 / win_rate | 盈利次数占交易次数比例。 |
| 盈亏比 | 平均盈利/平均亏损（若有）。 |
| 其他 | 如 Calmar、波动率等，以 evaluate 输出为准。 |

## 可视化

- **visual=True**：在 **qt.run(..., visual=True)** 时，会生成并展示资金曲线、回撤等图。
- 也可单独调用绘图接口（若有），传入结果对象得到相同或更多图表。

## 结果导出

- 将**资金曲线**、**交易列表**导出为 DataFrame 或 CSV：从结果对象中取出对应属性后使用 **to_csv()** 或 **to_excel()** 等。
- 便于在外部进行进一步分析或报告。

## 简要分析思路

- 结合 **trade_log** 与**绩效指标**：检查高回撤时段对应的交易、胜率与盈亏比的匹配度。
- 可对策略参数、标的池或区间做敏感性分析，或使用《优化交易策略》中的参数优化（mode=2）寻找更优参数。
